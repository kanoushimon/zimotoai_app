#地元あいアプリケーション

これは、ユーザーが新しい地元を探し、地元が大好きな人同士が繋がるためのアプリケーションです。
作者：狩野志文
作成日：2020年

#機能紹介
・ローカルメディアのマップ検索
・日替わりおすすめの地域とメディア
・登録ユーザーの投稿・交流機能
