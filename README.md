「地元あい」

地元あいは、ユーザーが新しい地元を探し、地元が大好きな人同士が繋がるためのアプリケーションです。
作者：狩野志文
作成日：2020年

#機能紹介
・ローカルメディアのマップ検索
・ランダム表示のおすすめの地域とメディア
・ユーザー登録後の投稿・交流機能
